Technology Used

  • HTML5
  • CSS3
  • Bootstrap 4.0 FrameWork For Responsive UI
  • JavaScript (70%)
  • ECMAScript (20%)
  • jQuery (10%)
  • JSON (YouTube Data API v3)
  • Font-Awesome Icon Library v4.7
  • Social Share Kit For Responsive Sharing Links

Validations

  • Empty URL
  • Wrong Url (Other than YouTube)
  • ‎Invaild Url (Url that is Genuine but includes No Video)
  • Server Errors:
    • Resource Not Found (404)
    • Internal Server Error (500)
    • Server TimeOut
    • Parsing Errors and Some Uncaught Errors

Exception Cases

  • Doesn't Include Video Owner Comment and Like In Random Comment Generation (Abviously Who is going to Conduct GiveAway Should Not Be A Winner)
  • Filter Out Every Comment and Like. Considers 1 Comment/Like Per User
  • ‎In Case there is No Comment having any Like on the video It Display Appropriate Message.
  • While Picking Max Liked Comment Case If there are more than two users have Same Number of Likes, project will Randomly Declare A Winner among them! So Every User Have Equal Chance Of Winning.
